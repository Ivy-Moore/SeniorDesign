Needed:
Flutter
Android Studio

To install Flutter:
https://flutter.io/docs/get-started/install
Note: DO NOT extract into /program files type directories
      Also add flutter/bin to path
      running 'flutter doctor' will help solve issues with flutter installation

Setting up Flutter on Android Studio:
      Open Android Studio and click on the settings icon and go to plugin
      Click on Browse repositories
      Search for Flutter and install it, also install Dart
      Restart
